📊 COVID-19 Exploratory Data Analysis Tracker

## 🧠 Project Overview

This project performs **Exploratory Data Analysis (EDA)** on COVID-19 datasets to understand global and country-level trends during the pandemic.

The main goal is to extract meaningful insights using data visualization and statistical exploration, helping to answer questions like:

* How did COVID-19 cases evolve over time?
* Which countries were most affected?
* What trends appear in deaths, recoveries, and active cases?
* How did the pandemic spread globally?

---

## 🚀 Key Features

✔ Data Cleaning & Preprocessing
✔ Time-series analysis of COVID-19 trends
✔ Country-wise comparison
✔ Visual analytics using graphs & plots
✔ Statistical exploration of pandemic patterns
